## Manage secrets

Usage: 
```bash
aloma secret [options] [command]
```

Options:
```bash
-h, --help                    display help for command
```

Commands:
```bash
list [options]                List all secrets
add [options] <name> <value>  Add a new secret
delete [options] <name>       Delete a secret
help [command]                display help for command
```
